# Vlad's dotfiles

## Installation

```bash
# Required VGS library
git clone https://github.com/vghn/vgs.git ~/vgs

# Dotfiles
git clone https://github.com/vladgh/dotfiles.git ~/dotfiles
~/dotfiles/install.sh
```
OR
```
PRIVATE_DIR=/path/to/secrets dotfiles/install.sh
```
