# Vlad's dotfiles

## Installation

```bash
git clone --recursive https://github.com/vladgh/dotfiles.git
dotfiles/bin/install.sh
# OR
PRIVATE_DIR=/path/to/secrets dotfiles/bin/install.sh
```
